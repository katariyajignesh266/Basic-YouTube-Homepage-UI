Basic YouTube Homepage UI
Project Overview

This project is a responsive clone of the YouTube Homepage User Interface developed using HTML and CSS. The main objective of this project is to practice modern web design principles, responsive layouts, and user interface development by recreating the appearance of YouTube's homepage.

The website includes a navigation header, sidebar menu, category section, search bar, video cards, and a responsive layout that adapts to different screen sizes.

Features
Responsive YouTube-inspired homepage design
Fixed navigation header
Search bar with modern styling
Sidebar navigation menu
Video category filter section
Video thumbnail cards
Channel information display
Mobile-friendly responsive layout
Clean and organized UI structure
Modern CSS styling and hover effects
